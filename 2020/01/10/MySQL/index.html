MySQL建表约束

主键约束

  • 它能够唯一确定一张表中的一条几率，也就是我们通过给某个字段添加约束，就可以使得该字段不重复且不为空

  • 关键字：PRIMARY KEY

  • 插入数据id不能相同和NULL

    create table 表名(
    id int PRIMARY KEY,
    name varchar(20)
);

  • 联合主键，插入数据只要联合的主键值加起来不重复就可以

    create table 表名(
    id int,
    name varchar(20),
    password varchar(10),
    primary key(id,name)
);

自增约束

  • 和primary key联合使用

  • 关健字：auto_increment

    create table 表名(
    id int primary key auto_increment,
    name varchar(20)
);

  • 建表时忘记创建主键

    alter table 表名 add primary key

  • 删除主键约束

    alter table 表名 drop primary key

唯一约束

  • 约束修饰的字段的值不可以重复

  • 关键字：unique

非空约束

  • 修饰的字段不能为空 NULL
  • 关键字：not null

默认约束

  • 当我们插入字段值的时候，如果没有传值，就会使用默认值

    create table 表名(
    id int,
    name varchar(20),
    age int default 10
);

外键约束

  • 涉及两个表：父表，子表

  • 主表中被引用的字段没有数据值，在副表中是不可以使用的

  • 主表中的记录被副表引用，是不可以被删除的

    • 班级
    create table classes(
    id int primary key,
    name varchar(20)
);
    • 学生
    create table students(
    id int primary key,
    name varchar(20),
    foreign key(class_id) references classes(id)
);

单表查询练习

  • 查询一张表的所有记录

    select * from students;

  • 查询一张表的指定字段的所有记录

    select sno,sname,class from students;

  • 查询一张表的指定字段的不重复记录

    select distinct sbirthday from students;

  • 查询一张表的指定字段数值区间的所有记录

    select * from students where sno between 2 and 3;
    select * from students where sno > 2 and sno < 4;

  • 查询一张表的指定字段满足数值的所有记录

    select * from students where sno in(2,3,4);
    select * from students where class="102" or sno="2";

  • 查询一张表中的以某个字段降序所有记录（默认升序）

    select * from students order by class desc;

  • 查询一张表中的以某个字段降序且以某个字段升序的所有记录

    select * from students order by sno asc,class desc;

  • 查询一张表中指定字段数值的所有记录数

    select count(*) from students where class="102";

  • 查询一张表中指定字段数值最大以及最小的记录

    select max(sno),min(sno) from students;

  • 查询一张表中指定指端数值的平均值

    select avg(degress) from score;
    select avg(degress) from score where cno="3-105";
    • 分组查询
    select cno,avg(degree) from score group by cno;

子查询

  • 查询某班级学生每门课的平均分

    第一步找某班级的所有学生
select * from students where class = "101";
第二步根据学生的对应的老师编号(科目)分组查询成绩
select * from score where sno in(select sno from students where class = "101");

最后查询结果
select cno,avg(degree) from score
where sno in(select sno from students where class = "101")
group by cno;

MySQL事务

  • 最小的不可分割的工作单元，能够保证一个业务的完整性。

  • 多条SQL语句执行，要么同时成功，要么同时失败。

  • MySQL默认是开启事务的。

    select @@autocommit;

  • 关闭自动提交

    set autocommit=0;

  • 事务开启三种方式

    set autocommit = 0;
    begin
    start transaction

  • 事务提交

    commit;

  • 事务回归

    rollback;

  • 事务四大特性：原子性、一致性、隔离性、持久性
